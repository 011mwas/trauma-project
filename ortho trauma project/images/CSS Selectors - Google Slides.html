Cascading Style Sheets (CSS)

A style sheet language used for describing the presentation semantics (the look and formatting) of a document written in a markup language. Its most common application is to style web pages written in HTML.

So who wins if multiple rules match?

  • !important wins
  • style attribute within element (inline CSS) wins
  • CSS Specificity scoring
    • 100 pts for ids
    • 10 pts for classes
    • 1 pt per tag
    • If the points tie, then the last one wins!
  • If no CSS rules match
    • DOM tree inherited text-related properties
    • User Agent

:: Pseudo Elements

Creates elements (or divides existing element) to use for selections. For instance, document languages do not offer mechanisms to access the first letter or first line of an element's content. Pseudo-elements allow designers to refer to this otherwise inaccessible information. Examples:

::first-letter

::first-line

::before

::after

Great link with examples:

http://www.w3.org/TR/2001/CR-css3-selectors-20011113/#pseudo-elements
